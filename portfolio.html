Game Development



The Dark Journey: An Exposure Therapy Game for Nyctophobia

The Dark Journey is an innovative 3D game designed as an exposure therapy tool. The game features progressively challenging tunnel environments with mild jump scares and eerie sound effects to help users build resilience in a controlled setting. Targeting teens and middle-aged adults, it provides a safe, immersive experience while excluding high-risk individuals. By simulating fear triggers gradually, this therapeutic game aims to reduce anxiety, improve coping mechanisms, and enhance quality of life for phobia sufferers.
